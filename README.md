# Agent Quality & Token Optimization — Workshop

A customer-facing workshop on getting **more quality and fewer wasted tokens** out of GitHub Copilot agents. Rebuilt from the deck as an interactive, zero-dependency web page.

**Live:** https://mdiiacovo.github.io/agent-quality-optimization/

## Delivering it live
- **← / →** (or **↑ / ↓ / PageUp / PageDown / Space**) — navigate
- **`n`** — toggle the speaker-notes panel (notes on nearly every slide)
- **`o`** — open the **Slides** panel to **show/hide slides** for your audience
- **`h`** — hide the current slide
- Hidden slides are skipped during the talk and remembered on your device ("Show all" resets)

## Contents
Three parts: **why agent quality is the better focus**, **foundations** (LLMs, agents, context windows), and **quality & token controls** (model choice, prompts, phases, deterministic guardrails, and agent configs) — closing with 5 things you can do today.
